## Primeiro Semestre – Análise e Desenvolvimento de Sistemas

No meu primeiro semestre, iniciei a jornada no mundo da programação e desenvolvimento de software.  
Estudei **Algoritmos**, aprendendo a pensar de forma lógica e estruturada para resolver problemas computacionais.  
Desenvolvi habilidades em **estruturação de dados, fluxogramas, pseudocódigo e lógica de programação**, criando programas simples e eficientes.  
Além disso, comecei a compreender conceitos fundamentais de **análise de sistemas e boas práticas de programação**, construindo uma base sólida para os semestres seguintes.
